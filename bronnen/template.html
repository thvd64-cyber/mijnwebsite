⚠️ Bij gebruik van deze template wordt bij import automatisch een unieke ID gegenereerd op basis van Doopnaam, Roepnaam, Achternaam en Geslacht, gevolgd door 3 cijfers. Je hoeft deze kolom dus niet zelf in te vullen.

Hoe werkt de CSV-export?
1. CSV-bestand kan geopend worden in Excel of andere spreadsheetprogramma's.
2. Je kunt altijd zelf een bestandsnaam kiezen via de prompt.
3. DuckDuckGo, Chrome of Edge (HTTPS of localhost): echte “Opslaan als” dialoog.
4. Andere browsers (Firefox, Safari, oudere versies): locatiekeuze niet ondersteund. Alert legt uit.
5. Maximaal compatibel en duidelijke feedback ongeacht browser.
