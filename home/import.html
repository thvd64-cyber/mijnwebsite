Import
Hoe werkt de CSV-import?
1. Je kunt een CSV-bestand importeren dat is geëxporteerd uit deze applicatie of een correct opgemaakte stamboom CSV.
2. Alleen CSV-bestanden worden ondersteund. Andere formaten worden niet ingelezen.
3. De import zal bestaande stamboomgegevens **overschrijven** of **toevoegen**, afhankelijk van de implementatie van je import.js.
4. Het laden gebeurt **lokaal in je browser**; er wordt niets online verstuurd, dus je gegevens blijven privé.
5. Browserverschillen:
  - DuckDuckGo, Chrome, Edge en moderne browsers ondersteunen grote bestanden en geven meestal directe feedback bij fouten.
  - In oudere browsers kan de performance beperkt zijn bij zeer grote CSV-bestanden.
